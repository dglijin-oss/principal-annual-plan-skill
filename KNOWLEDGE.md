## 一、学校管理核心理论

### 1. PDCA循环（戴明环）
- **Plan（计划）**：制定目标和行动方案
- **Do（执行）**：按计划实施
- **Check（检查）**：评估执行效果
- **Act（改进）**：根据检查结果调整

### 2. 目标管理（MBO）
- 目标要符合SMART原则：具体(Specific)、可衡量(Measurable)、可达成(Achievable)、相关性(Relevant)、有时限(Time-bound)
- 每个目标都要有责任人、时间节点、考核标准

### 3. SWOT分析
- **S（优势）**：学校的核心竞争力
- **W（劣势）**：需要改进的方面
- **O（机会）**：外部环境的有利因素
- **T（威胁）**：外部环境的挑战

### 4. 教育质量管理
- 全面质量管理（TQM）在学校管理中的应用
- 以学生为中心的教育理念
- 持续改进的质量文化

## 二、K12学校年度计划标准框架

### 核心模块
1. **教学工作**：课程建设、教学改革、质量监控、教研科研
2. **德育工作**：思想教育、行为规范、心理健康、家校共育
3. **师资建设**：培训发展、考核评价、梯队建设、激励机制
4. **学生发展**：学业进步、身心健康、特长培养、综合素质
5. **校园文化**：校风学风、特色活动、品牌建设、社区关系
6. **安全管理**：校园安全、食品安全、心理健康、应急预案
7. **财务后勤**：预算管理、硬件改善、食堂宿舍、信息化

### 时间线
| 时间 | 重点工作 |
|------|---------|
| 1月 | 上年度总结、下年度计划制定 |
| 2月 | 开学准备、教师培训 |
| 3月 | 教学计划执行、教研活动启动 |
| 4月 | 期中考试、质量分析 |
| 5月 | 校园文化活动、家长开放日 |
| 6月 | 期末考试、招生宣传 |
| 7-8月 | 暑期培训、硬件维修、新教师入职 |
| 9月 | 新学期开学、安全教育月 |
| 10月 | 教学检查、教师评估 |
| 11月 | 期中考试、家长会 |
| 12月 | 年度总结、财务结算 |

## 四、常见问题与应对策略

### 问题1：计划写了但执行不下去
**原因**：目标太高、措施太虚、责任不清、没有检查
**对策**：
- 目标要"跳一跳够得着"
- 每项措施必须责任到人
- 建立月度/季度检查机制
- 将执行情况纳入绩效考核

### 问题2：各部门各写各的，缺乏协同
**原因**：没有统一的框架和沟通机制
**对策**：
- 制定统一的计划模板
- 召开计划协调会，让各部门负责人一起讨论
- 设置跨部门协作的项目和指标

### 问题3：计划与实际脱节
**原因**：闭门造车，不了解一线实际情况
**对策**：
- 计划制定前做一线调研
- 征求教师代表和家长委员会意见
- 预留弹性空间应对变化

### 问题4：预算不够怎么办
**对策**：
- 优先保障教学和安全的预算
- 寻找社会资源（校友捐赠、企业合作）
- 分阶段实施，先做投入小见效快的项目
