# 校长专属AI · 年度工作计划生成与优化

> 为校长和学校管理者打造的AI年度工作计划工具。

## 功能

- ✅ 自动生成学校年度工作计划
- ✅ 覆盖六大模块：教学/师资/学生/文化/安全/财务
- ✅ 智能优化已有计划
- ✅ 从工作总结生成下年计划
- ✅ 提供量化指标参考
- ✅ 季度复盘时间表生成

## 适用场景

- K12学校年度工作计划制定
- 学期工作计划编制
- 年度工作总结与下年规划
- 学校发展规划
- 预算与资源分配

## 使用方式

将 `SKILL.md` 放入你的 AI 系统的 skills 目录即可使用。

## 文件结构

```
principal-annual-plan/
├── SKILL.md          # 主技能文件
├── KNOWLEDGE.md      # 专业知识库
└── README.md         # 项目说明
```
